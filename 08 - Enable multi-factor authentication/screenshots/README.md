# Lab 08 - Enable Azure AD Multi-Factor Authentication

This lab demonstrates how to enable Azure AD Multi-Factor Authentication (MFA) using Conditional Access and per-user settings in Microsoft Entra ID. You will configure a Conditional Access policy that requires MFA and also explore enabling MFA at the user level.

## Screenshots

- `conditional-access-grant-require-mfa.png`: Configuring the “Grant” control in a Conditional Access policy to require MFA for access.
- `select-user-for-policy.png`: Selecting a user in Microsoft Entra ID to apply the Conditional Access policy requiring MFA.
- `mfa-settings-overview.png`: Overview of the MFA configuration page under the Microsoft Entra admin center.
